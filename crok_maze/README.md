# crok maze

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a maze like structure.

Setup:

    - Width: wall width
    - Zoom: zoom in | out of the maze
    - Rotation: rotates the maze
    - Position: offset the maze position
