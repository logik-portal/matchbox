# crok wave lines

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a wave line pattern.

Setup:

    - Lines: number of waves
    - Speed: speed of the waves
    - Offset: offset lines in time
    - Brightness : brightness of the lines
    - Glow: overall glow
    - Colour: colour for the 4 gradient points
