# CPGP Flame

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Generates a simple moving flame.
You can change the color of the base and trail of it.

Since we can't have RGB in the GUI yet, X stands for R, Y for G, Z for B.
