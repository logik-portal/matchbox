# crok cameraflash

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates simple camera flashs.

Setup:

    - Flickering: amount of flickering
    - Density: amount of flashes
    - Size: size of the flashes
    - Seed: change seed value
