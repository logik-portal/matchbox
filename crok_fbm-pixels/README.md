# crok fbm-pixels

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a fbm style pattern.

Setup:

    - Scale: adjust scale of the pattern
    - Detail: adjust pattern detail
    - Speed: adjust pattern animation speed
    - Offset: offset pattern in time
