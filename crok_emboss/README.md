# crok emboss

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Simulates an emboss effect.

Input:

    - Front: source clip

Setup:

    - Height: bump height
    - Specular: adjust the glossiness
    - Light Height: height of the lightsource
    - Position: light position in x / y
