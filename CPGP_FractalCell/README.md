# CPGP FractalCell

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Generates a fractal cell texture.

Parameters:

    - Cell Speed: change the speed of the Cell generation, to get more or less 'life'
    - Use Light Modulation : adds flickering
