# K BW

**Author:** Kyle Obley

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a black and white image based on the dominance of R-G-B channels.

Common values to mimic specific filters:

    Yellow: 60 28 12
    Orange: 78 22 0
    Red: 90 10 0
    Green: 10 70 20
    Infrared: 40 140 -80
