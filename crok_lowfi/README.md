# crok lowfi

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Simulates NES, EGA and Gameboy video out.

Setup:

    - Brightness: adjust brightness of the image
    - NES: apply NES look
    - EGA: apply EGA colour palette
    - GAMEBOY: apply a gameboy type colour pallete
