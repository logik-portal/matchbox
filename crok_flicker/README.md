# crok flicker

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a flickering effect.

Input:

    - Front: source clip

Setup:

    - Amplitude: amplitude of the flickering
    - Frequency: speed of the flickering
    - Seed: offset the random generator
