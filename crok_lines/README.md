# crok lines

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates lines.

Setup:

    - Density: amount of lines
    - Sharpness: edge sharpness
    - Itterations: Itterations
    - Speed: how fast the lines are moving
    - Offset: offset the time
    - Colour: color of the lines
