# crok pixelate

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Pixelates the image.

Input:

    - Front: source clip
    - Matte: matte to define the affected area

Setup:

    - Detail: detail of the pattern
    - Dissolve: blend between original and pixelated image
