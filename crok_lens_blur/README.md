**Action:** No

**Timeline:** No

**Transition:** No

## Description
Creates a Lens Blur effect.

Input:

    - Front: source clip

Setup:

    - Amount: how much lens Blur is applied
    - Aspect: aspect ratio
    - Chroma Offset: how much chromatic abberation
    - Gain: boost the incomming gain
    - Threshold: threshold amount for the gain
