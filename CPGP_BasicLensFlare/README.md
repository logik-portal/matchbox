# CPGP BasicLensFlare

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a cheap lens flare.
Inputs:

    - Front Noise: a noise texture

Params:

    - Direction : the direction of lens flare
    - Move: Use automated movement
    - Speed: the speed for movement
