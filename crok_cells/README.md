# crok cells

**Author:** Ivar Beer

**Shader Type:** Matchbox

**Action:** No

**Timeline:** No

**Transition:** No

**Minimum Flame Version:** 


## Description
Creates a cell pattern.

Setup:

    - Itterations: Itterations
    - Zoom: zoom in / out of the pattern
    - Speed: how fast the cells are moving
    - Offset: offset the time
    - Colour: color of the cells
