**Action:** Yes

## Description
Gives you Resolve style 'Sat vs Sat' curve controls.

Input:

    - Front: front image

Setup:

    - Sat Multiplier: multiplies the saturation of the curves to give you the possibility to go over 1.0 Saturation.
